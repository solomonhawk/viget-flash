# viget-flash

A flash card game for Viget employees!

### Requirements

- [node.js](http://nodejs.org/)

**Dev requirements:**

- [TJ Holowaychuk's fork of jscoverage](https://github.com/visionmedia/node-jscoverage)

### Build it

`npm install -d`

### Test it

`npm test`

### Check test coverage

`make coverage`

### Start it

`npm start`

### Compile assets

`grunt`

or

`grunt watch`

### Host it

``` shell
heroku create
git push heroku master
```
